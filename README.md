# NeoSlide 🧩

A futuristic, high-performance sliding tile puzzle game built with React, Vite, and Express. NeoSlide features fluid animations, multiple themes, and full PWA (Progressive Web App) support for a seamless "app-like" experience on mobile and desktop.

![NeoSlide Icon](/public/icon.svg)

## 🚀 Features

- **Futuristic UI**: Cyberpunk, Retro, and Minimal themes with fluid displacement effects.
- **Level System**: Progress through increasingly difficult puzzles with a visual level map.
- **PWA Ready**: Installable on Android, iOS, and Desktop with offline support.
- **Responsive Design**: Optimized for touch and keyboard controls across all devices.
- **Game Mechanics**: Includes hints, lives, scoring, and persistent save states.
- **QR Sharing**: Easily share your progress or the game via built-in QR code generation.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [Vite 6](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend**: [Express](https://expressjs.com/) (serving static assets and PWA files)
- **PWA**: [Vite PWA Plugin](https://vite-pwa-org.netlify.app/)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/neoslide.git
   cd neoslide
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   npm start
   ```

## 📱 PWA Support

NeoSlide is a fully compliant Progressive Web App. To install it:
- **iOS**: Open in Safari, tap the Share icon, and select "Add to Home Screen."
- **Android**: Open in Chrome and tap the "Install App" prompt or select it from the menu.
- **Desktop**: Click the install icon in the browser's address bar.
